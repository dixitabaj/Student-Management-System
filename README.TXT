## Teacher Management System (Extension Module)

As an extension of the Umbra architecture, a **Teacher Management System** module is included. This system is designed for academic institutions to manage teacher records efficiently using Java Swing and core data structures.

### Features

- Add, view, update, and delete teacher records
- Input validation for teacher details 
- Simple, clean GUI for smooth navigation
- In-memory storage using `ArrayList`
